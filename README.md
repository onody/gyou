# Gyou

ひらがなの文字が何行かを取得します。


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gyou'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gyou

## Usage

```ruby
gyou_sample = Gyou.get("し")
# gyou_sample = "さ"
gyou_sample = Gyou.get("すずき")
# gyou_sample = "さ"
```
